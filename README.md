# Smart Tourism AI

An AI powered tourism recommendation system built using **Python** and **Streamlit** that helps travelers discover destinations, analyze tourism data, generate personalized itineraries, and visualize travel routes across Tamil Nadu.


# Project Overview

Smart Tourism AI is an intelligent travel planning application that simplifies trip planning by recommending tourist destinations based on:

*  Source Location
*  Destination District
*  Budget
*  Trip Duration

The application filters destinations, provides AI powered travel insights, generates a travel itinerary, displays tourism analytics, and visualizes the travel route on an interactive map.


# Features
## Smart Dashboard

* Modern responsive UI
* Interactive sidebar filters
* Tourism recommendations
* Travel summary cards


## Destination Recommendation

* Filter places by district
* Budget based recommendations
* Highest rated tourist attractions
* Accommodation suggestions
* Weather information
* Best visiting month


## AI Trip Suitability Score

The application calculates a trip suitability score based on the selected budget and provides an overall recommendation.


## AI Travel Insights

Provides intelligent travel information including:

* Best tourist place
* Recommended travel month
* Accommodation
* Weather
* Best travel route


## Smart Itinerary Planner

Automatically generates a simple day wise itinerary for:

* Day 1
* Day 2
* Day 3


## Interactive Route Map

Displays:

* Source location
* Destination
* Route between locations
* Interactive OpenStreetMap visualization using Folium

  
## Tourism Analytics

Displays:

* Total recommended places
* Average rating
* Number of categories
* Number of hotels


## Tourist Places Table

Interactive data table containing:

* Place Name
* Category
* Rating
* Budget
* Weather
* Best Visiting Month
